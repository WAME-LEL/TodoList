## 실행방법

본 프로젝트는 React native로 만들어 졌으며 expo cli를 사용해서 만들었습니다.

만약 android studio와 같은 에뮬레이터를 실행시킬수 있는 환경이 있다면 에뮬레이터로 실행시켜도 되고
